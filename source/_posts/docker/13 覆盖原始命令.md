如果你的 Dockerfile 中有如下定义：
```
ENTRYPOINT ["python"]
CMD ["app.py"]
```
你可以在启动容器时使用 --entrypoint 选项来覆盖 ENTRYPOINT，并且提供新的命令来覆盖 CMD：
```
docker run --entrypoint /bin/sh <image>
```
或者同时覆盖 ENTRYPOINT 和 CMD：
```
docker run --entrypoint /bin/sh <image> -c "ls -la"
```
